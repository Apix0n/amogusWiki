❓ Un problème ?

Si:

  • vous voulez améliorer le projet,
  • vous voulez demander le changement de nom d'un personnage,
  • vous avez trouvé un personnage deux fois,
  • vous avez une question,
  • vous voulez supprimer un personnage,
