Choisissez comment vous voulez créer votre personnage:

Vous devez choisir comment vous allez créer votre personnage. Vous avez le choix parmi 4 options !

  • 🎨 Dessiner sur l'image originale
  • 🤺 Créer un personnage customisé
  • 🖼 Créer un arrière-plan customisé
  • 💡 Donner une idée

Vous pouvez également combiner plusieurs techniques pour créer un personnage encore plus original.

Exemples: cactus, fishus, icecreamus

❗ Chaque personnage doit avoir une visière, sinon il sera refusé.
